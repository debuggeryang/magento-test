# Magento Test

#### Version 0.0.1

This respository contains a few extensions that I need for a Magento2 test.

Most extension I have done are in [app](https://github.com/debuggeryang/magento-test/tree/development/app) directory.

To install this Magento, please clone this repo and and run `compser install` and then `bin/magento setup:install`
